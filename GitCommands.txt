
Git commands you will use
as of 25 October 2020


NEW FILE:
git add new-file
git commit new-file -m "Message you want to go with the new file"

EXISTING FILE YOU CHANGE:
git commit changed-file -m "Message you want to go with the changed file"

PUSH RESULTS UP TO GITHUB.COM
git push -u https://GitHub.com/Dwayne-Phillips/julia


SYNCHRONIZE THE LOCAL REPOSITORY TO THE REMOTE ONE
git pull

NOTE: WHEN YOU MAKE ANY CHANGE OR ADD ANY FILE TO THE LOCAL
...do the NEW FILE commands above.
